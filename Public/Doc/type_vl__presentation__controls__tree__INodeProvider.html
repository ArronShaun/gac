INodeProvider Class

Represents a node.

Member Functions:

NameDescription
CalculateTotalVisibleNodesCalculate the number of total visible nodes of this node. The number of total visible nodes includes the node itself, and all total visible nodes of all visible sub nodes. If this node is collapsed, this number will be 1.
GetChildCountGet the number of all sub nodes.
GetExpandingGet the expanding state of this node.
GetParentGet the parent node.
ReleaseChildRelease an instance of a sub node.
RequestChildRequest the instance of a specified sub node. If the sub node is not longer needed, a call to INodeProvider::ReleaseChild is required.
SetExpandingSet the expanding state of this node.
