GuiListControl Class

Represents a list control. A list control automatically read data sources and creates corresponding data item control from the item template.

Base Types:
GuiScrollView

Sub Types:

NameDescription
IItemArrangerItem arranger for a GuiListControl. Item arranger decides how to arrange and item controls. When implementing an item arranger, GuiListControl::IItemArrangerCallback is suggested to use when calculating locations and sizes for item controls.
IItemArrangerCallbackItem arranger callback. Item arrangers use this interface to communicate with the list control. When setting positions for item controls, functions in this callback object is suggested to call because they use the result from the GuiListControl::IItemCoordinateTransformer.
IItemCoordinateTransformerItem coordinate transformer for a GuiListControl. In all functions in this interface, real coordinate is in the list control's container space, virtual coordinate is in a space that the transformer created.
IItemPrimaryTextViewPrimary text view for GuiListControl::IItemProvider.
IItemProviderItem provider for a GuiListControl.
IItemProviderCallbackItem provider callback. Item providers use this interface to notify item modification.
IItemStyleControllerItem style controller for a GuiListControl. Item style controller contains all information to render a binded item.
IItemStyleProviderItem style provider for a GuiListControl. When implementing an item style provider, the provider can require the item provider to support a specified view to access data for each item.
KeyDirectionRepresents the four directions that is accessable by keyboard.
